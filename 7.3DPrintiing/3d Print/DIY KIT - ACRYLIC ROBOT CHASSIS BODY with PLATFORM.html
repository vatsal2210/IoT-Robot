Package Contents
 
1 x Acrylic Robotic Base  
1 x Acrylic Platform 
2 x BO Motor 
2 x BO Motor Wheel 
2 x BO Motor Clamp 
1 x Castor Wheel 
4 x Sleeve (40mm) 
6 x Sleeve (15mm) 
7 x Screw + Nuts (M3 x 50mm) 
8 x Screw + Nuts (M3 x 25mm) 
2 x Pointed Screw (M2 x 10mm)
NO ELECTRONIC CIRCUIT BOARDS INCLUDED
 
(SUITABLE FOR MAKING FULL ACRYLIC ROBOT CHASSIS WITH PLATFORM)
 
NOTE : THIS IS A DIY(DO IT YOURSELF) KIT AND WILL NEED TO BE ASSEMBLED. 
 
IF YOU HAVE ANY DOUBTS, PLEASE CONTACT US BEFORE PAYING FOR THE ITEM.
 
 
Product Details
 
Use 40mm Sleeve as spacers on M3x50mm Screws for mounting Platform. 
Use 2 of 15mm Sleeves on each of M3x50mm Screws for Castor Wheel.
Use M3x25mm Screws for fixing BO Motor Clamps as well as BO Motors onto it. 
Use Pointed Screws for fixing BO Motor wheels.
2 Holes for connecting wires from BO Motors.
 
Users can drill extra holes using drillers or use two sided tapes for fixing the boards as per required.
